# styleark

Atomic CSS Classes

## Usage

Include the distribution styles file in the head tag of your html:

   <html>
     <head>
       <title>Example</title>
       <link rel="stylesheet"
         href="https://unpkg.com/@knowark/styleark@0.1.2/dist/styles.min.css">
     </head>
     <body></body>
   </html>
